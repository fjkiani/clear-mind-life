## Code Intelligence Tools

The Code Intelligence section of the dashboard gives you direct access to the five coding databases that power the Billing Agent. Use these tools to look up codes, check bundling rules, and verify billing limits before you submit.

## ICD-10 Lookup

Search by code or description. Enter "F32" to see all major depressive disorder codes. Enter "anxiety" to see all anxiety-related diagnoses. The search returns the code, full description, and valid billing indicator.

**Specificity matters.** Many payers require the most specific code available. Billing F32 (major depressive disorder, unspecified) when F32.1 (moderate) or F32.2 (severe without psychotic features) is documented will trigger a denial for insufficient specificity. The lookup tool shows you the full code hierarchy so you can select the right level.

## NCCI PTP Checker

Enter two CPT codes to check whether they can be billed together. The checker returns: whether the pair is flagged, the modifier indicator, and the effective date of the edit.

**Modifier indicator 0** means no modifier can override the edit — the codes cannot be billed together under any circumstances. **Modifier indicator 1** means a modifier (typically -25 or -59) can override the edit if the services were genuinely separate.

### What Modifier -25 Means

Modifier -25 is appended to an E&M code (99213, 99214, 99215) when a procedure is also performed on the same day. It tells the payer: the evaluation and management service was a significant, separately identifiable service — not just the pre-work for the procedure.

Without modifier -25, the payer bundles the E&M into the procedure fee and denies the E&M line. This is one of the most common denial causes in mental health practices that also provide medication management or minor procedures.

## CPT RVU Lookup

Search by CPT code to see the Relative Value Unit breakdown and national payment rate.

Each CPT code has three RVU components:
- **Work RVU:** The physician's time, skill, and effort
- **Practice Expense RVU:** Overhead costs (staff, equipment, supplies)
- **Malpractice RVU:** Professional liability insurance allocation

The total RVU multiplied by the Medicare conversion factor ($33.89 for 2025) gives the national payment rate. Commercial payers typically pay a percentage of Medicare rates — your contracted rate may be 110%, 120%, or higher depending on your payer contracts.

Use the RVU lookup to verify that your billed charges are in a reasonable range for the code. Significantly underbilling relative to the RVU-based rate is a common issue in mental health practices.

## MUE: Maximum Units Per Day

Medically Unlikely Edits define the maximum number of units that can be billed for a CPT code on a single date of service for a single patient. Billing above the MUE triggers an automatic denial.

The MUE for CPT 90837 (60-minute psychotherapy) is 1. You cannot bill two units of 90837 for the same patient on the same day. If a patient has two separate therapy sessions in one day, the second session should be billed with a different CPT code or with an appropriate modifier and documentation.

## Using Code Intelligence Before Submission

The Billing Agent runs all of these checks automatically on every claim. The Code Intelligence tools are for manual lookups — when you want to verify a code before the encounter, check a bundling rule you're unsure about, or research a denial reason code.

If you receive a denial with reason code CO-4 (modifier required), use the NCCI PTP checker to identify which modifier is needed. If you receive CO-97 (payment included in allowance for another service), the PTP checker will show you which code pair triggered the bundling edit.
